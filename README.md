# First Extension For HBuilderX

经常用到bootcdn的看过来~

## 功能特性

* 一键插入链接
* 支持快捷键(`Ctrl+Shift+I`)(`HBuilderX 3.1.22+`)
* 支持本地缓存，快速加载
